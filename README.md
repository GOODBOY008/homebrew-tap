# Homebrew Tap for r-shell

This is a [Homebrew](https://brew.sh/) tap for [r-shell](https://github.com/GOODBOY008/r-shell) and related applications.

## Installation

```bash
# Add this tap
brew tap GOODBOY008/tap

# Install r-shell
brew install --cask r-shell
```

## Updating

```bash
# Update Homebrew
brew update

# Upgrade r-shell
brew upgrade --cask r-shell
```

## Uninstallation

```bash
# Uninstall r-shell
brew uninstall --cask r-shell

# Remove all app data
brew uninstall --zap --cask r-shell
```

## Available Casks

- **r-shell** - Modern SSH terminal client built with Tauri and React

## Maintenance

This tap is automatically updated by GitHub Actions when new releases are published.
